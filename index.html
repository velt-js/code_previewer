Welcome to Code Preview

To use this tool, please provide the following URL parameters:

  • github: URL of the GitHub repository
  • preview: URL of the preview page
  • hideToolbar (optional): Set to 'true' to hide the toolbar
  • selectedTab (optional): Set to 'Code' or 'Preview' to select the initial tab

Example URL:

https://yourpage.com?github=https://github.com/user/repo&preview=https://preview-url.com&hideToolbar=true&selectedTab=Code
